# Academic Credential Verification System (ACVS)

## Features

- Issue credentials using blockchain
- Verify credentials
- QR code verification
- File upload (certificate)
- MySQL database integration

## Setup Instructions

### Backend

```
cd blockchain-backend
npm install
node server.js
```

### Frontend

```
cd credential-frontend
npm install
npm start
```

## 🛠 Tech Stack

- React.js
- Node.js / Express
- MySQL
- Ethers.js
